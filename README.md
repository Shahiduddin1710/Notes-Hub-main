##  Getting Started

### Clone Repository

```bash
git clone https://github.com/Shahiduddin1710/Notes-Hub-main.git
cd NotesHub
```

---

### Install Dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### Run Application

#### Start Backend

```bash
npm start
```

#### Start Frontend

```bash
npm start
```
